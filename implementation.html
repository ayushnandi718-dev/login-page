🚀 Implementation Guide

Follow these steps to integrate the Auth UI Template into your project.

1. Clone the Repository

git clone https://github.com/your-username/login-page.git
cd login-page

2. Setup Configuration

Rename the config file:

/config/config.example.js → /config/config.js

Add your API keys:

export const CONFIG = {
  GOOGLE_CLIENT_ID: "YOUR_GOOGLE_CLIENT_ID",
  GITHUB_CLIENT_ID: "YOUR_GITHUB_CLIENT_ID",
  FACEBOOK_APP_ID: "YOUR_FACEBOOK_APP_ID",
  LINKEDIN_CLIENT_ID: "YOUR_LINKEDIN_CLIENT_ID",
  REDIRECT_URI: "http://localhost:5500/callback/linkedin.html"
};

3. Enable Authentication Providers

  • Google → Firebase or Google Cloud
  • GitHub → Developer Settings
  • Facebook → Meta Developers
  • LinkedIn → LinkedIn Developers

4. Run the Project

Open index.html in your browser

5. LinkedIn Setup (Important)

LinkedIn requires backend integration.

⚠️ The template only handles UI + redirect.
You must exchange the code on your backend.

6. Deploy

You can deploy using:

  • GitHub Pages
  • Netlify
  • Vercel
